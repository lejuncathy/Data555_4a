# Measles Dynamics Dashboard

## Project overview
This dashboard compares simulated measles epidemic dynamics with observed historical measles case counts from London.

## Dashboard link
[View dashboard here](https://lejuncathy.github.io/Data555_4a/)

## Data
- Simulated TSIR output
- Historical London measles data from the `tsiR` package

## Methods
The dashboard was created in R using `flexdashboard`, `plotly`, `dplyr`, and `tsiR`.

## Why this project matters
Understanding measles transmission patterns can help researchers and public health professionals evaluate how epidemic dynamics evolve over time.  
By comparing simulated and observed data, this dashboard provides an accessible way to interpret infectious disease behavior and assess how well simple transmission models reflect real outbreaks.
